系统能力和权限说明

本应用的核心功能在设备本地完成。当前版本不涉及账号登录、联网同步、定位、通讯录、相机、麦克风、日历、短信、通话记录等能力。

应用会在你点击“一键复制”时调用系统剪贴板能力，将你主动选择的 Prompt 内容写入设备剪贴板，便于粘贴到其他应用。应用不会读取其他应用写入的剪贴板内容，也不会上传剪贴板内容。

应用会在你主动导出或导入备份时调用系统文件选择或保存能力。导出的 JSON 备份文件只保存到你选择的位置；导入时，应用只读取你主动选择的备份文件，并将其中可识别的数据合并到当前本地数据中。

数据存储、导出、导入和删除

  • 本地存储：Prompt、模板、分类、收藏、历史记录、工作流、评分摘要、草稿和设置保存在当前设备本地。
  • 数据导出：你可以在设置页主动导出 JSON 备份文件，导出内容包括自建模板、分类、历史记录、工作流、内置模板收藏状态和必要设置。
  • 数据导入：你可以在设置页主动选择备份文件并合并导入。导入会尽量保留现有数据，并对重复或冲突内容进行跳过、合并或重命名处理。
  • 数据删除：你可以在应用内清理历史记录，或在模板、分类、工作流等对应页面管理自建内容。你也可以通过系统设置清理应用数据或卸载应用来删除本地数据。

System Capabilities and Permissions

The core features of the app are completed locally on the device. The current version does not involve account login, online synchronization, location, contacts, camera, microphone, calendar, SMS, call records, or similar capabilities.

When you tap "one-tap copy", the app uses the system clipboard capability to write the Prompt content you actively selected to the device clipboard, so you can paste it into other apps. The app does not read clipboard content written by other apps and does not upload clipboard content.

When you actively export or import a backup, the app uses the system file selection or save capability. Exported JSON backup files are saved only to the location you choose. During import, the app reads only the backup file you actively selected and merges recognizable data into the current local data.

Data Storage, Export, Import, and Deletion

  • Local storage: Prompts, templates, categories, favorites, history, workflows, scoring summaries, drafts, and settings are stored locally on the current device.
  • Data export: You can actively export a JSON backup file from the settings page. The export may include custom templates, categories, history, workflows, built-in template favorite status, and necessary settings.
  • Data import: You can actively select a backup file from the settings page and merge-import it. Import attempts to preserve existing data and may skip, merge, or rename duplicate or conflicting content.
  • Data deletion: You can clear history in the app or manage custom content on the relevant template, category, workflow, and other pages. You can also delete local data by clearing app data in system settings or uninstalling the app.

User Rights

You can view, copy, edit, export, or delete locally saved content within the app. Because the current version mainly uses local storage, these operations need to be completed on your device.

Third-Party Services and Information Sharing
