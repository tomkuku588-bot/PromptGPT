# PromptGPT 协议托管页

本目录是面向 GitHub Pages 的静态站点，用于托管 PromptGPT / GPT提示词助手 的隐私政策与用户协议。

## 文件

- `index.html`：中英文切换的单页协议中心，包含隐私政策和用户协议。
- `privacy-policy.html`：兼容旧链接，自动跳转到 `index.html#privacy`。
- `user-agreement.html`：兼容旧链接，自动跳转到 `index.html#terms`。

## GitHub Pages 地址

- 首页：`https://tomkuku588-bot.github.io/PromptGPT/`
- 隐私政策：`https://tomkuku588-bot.github.io/PromptGPT/#privacy`
- 用户协议：`https://tomkuku588-bot.github.io/PromptGPT/#terms`

## 部署方式

1. 将本目录作为仓库根目录推送到 `git@github.com:tomkuku588-bot/PromptGPT.git` 的 `main` 分支。
2. 在 GitHub 仓库 Settings -> Pages 中选择 `Deploy from a branch`。
3. 发布源选择 `main` 分支和 `/root` 目录。
