# PromptGPT Pages

PromptGPT / GPT 提示词助手的 GitHub Pages 静态协议页面仓库。

## 推荐入口

- `index.html`: 中英文单页协议中心，包含隐私政策和用户协议。
- `privacy-policy.html`: 隐私政策兼容入口，跳转到 `index.html#privacy`。
- `user-agreement.html`: 用户协议兼容入口，跳转到 `index.html#terms`。

## 公开链接

- 首页：`https://tomkuku588-bot.github.io/PromptGPT/`
- 隐私政策：`https://tomkuku588-bot.github.io/PromptGPT/#privacy`
- 用户协议：`https://tomkuku588-bot.github.io/PromptGPT/#terms`
